# Sprint7
Este es una repositorio para poder realizar un proyecto del Sprit 7 de cientifico de datos en el bootcamp de tripleten

Este proyecto es una aplicación web interactiva desarrollada con Streamlit para el análisis y visualización de datos, utilizando Plotly para la generación de gráficas dinámicas.
El objetivo es facilitar la exploración de datos de forma visual, clara y accesible desde el navegador.


Interfaz web sencilla e intuitiva

Visualizaciones interactivas con Plotly

Carga y procesamiento de datos con Python

Gráficas responsivas adaptadas al tamaño de la pantalla

Ejecución local o en la nube (Streamlit Cloud)

Tecnologías utilizadas

Python 3.x

Streamlit

Plotly

Pandas

Objetivo del proyecto

Este proyecto fue desarrollado con fines educativos y prácticos, para reforzar habilidades en:

Análisis de datos

Visualización interactiva

Desarrollo de aplicaciones web con Python
